
# Introduction

Project to simple test performances between different app urls.

# Setup

```bash
# Set env variables
cp .env.example .env
# Install
npm install
# Run
node src/index.js
```

Set the apps and URLs in `index.js` like example.

Number of requests of each iteraction and number of iteractions can be set on .env.
